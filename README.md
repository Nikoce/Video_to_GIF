# 视频转 GIF 网页版

这是给 Mac / Windows 同事使用的纯静态网页工具。视频在浏览器本地读取、抽帧和编码，不上传服务器。

## 使用

直接打开：

```text
web/video-to-gif/index.html
```

操作流程：

1. 拖入或选择视频。
2. 设置开始时间、结束时间、帧率、最大宽度和预设。
3. 点击“生成 GIF”。
4. 预览结果后点击“下载 GIF”。

## 当前边界

- 推荐截取 3-8 秒片段，过长片段会比较慢。
- 浏览器能不能读取视频取决于本机浏览器的视频解码能力。Mac 上建议使用 MP4/H.264。
- 第一版使用浏览器端 256 色 GIF 编码器，优先保证免安装、免上传和跨平台使用。
- 如果后续要求更高画质、更小体积或批量处理，可以升级为 ffmpeg.wasm 或服务端处理方案。
