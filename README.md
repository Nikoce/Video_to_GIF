# 视频转 GIF 网页版

这是给 Mac / Windows 同事使用的纯静态网页工具。视频在浏览器本地读取，并通过 ffmpeg.wasm 转换成 GIF，不上传服务器。

## 使用

直接打开：

```text
web/video-to-gif/index.html
```

操作流程：

1. 拖入或选择视频。
2. 拖动“片段截选”的开始点和结束点，或播放到当前画面后一键设为开始/结束。
3. 选择压缩方案，默认和本地工具一致使用“均衡清晰”。
4. 点击“生成 GIF”。
5. 预览结果后点击“下载 GIF”。

## 当前边界

- 推荐截取 3-8 秒片段，过长片段会比较慢。
- 浏览器能不能读取视频取决于本机浏览器的视频解码能力。Mac 上建议使用 MP4/H.264。
- 首次生成会下载 ffmpeg.wasm 运行组件，加载完成后再开始转换。
- 当前单文件处理优先；如果后续要批量处理，可以再按本地工具的批量队列继续扩展。
