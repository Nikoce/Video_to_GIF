# 视频转 GIF 网页版

这是给 Mac / Windows 同事使用的纯静态网页工具。视频在浏览器本地读取，并通过 ffmpeg.wasm 转换成 GIF，不上传服务器。

## 使用

直接打开：

```text
web/video-to-gif/index.html
```

操作流程：

1. 拖入或选择视频。
2. 默认会整段生成 GIF；需要截取时，打开“截取片段”。
3. 拖动“片段截选”的开始点和结束点，或播放到当前画面后一键设为开始/结束。
4. 多个视频时，可以勾选卡片后把当前整段/截取设置批量套用到选中视频。
5. 选择压缩方案，默认和本地工具一致使用“均衡清晰”。
6. 点击“全部生成 GIF”。
7. 在任务列表里预览或下载每个 GIF。

## 当前边界

- 推荐截取 3-8 秒片段，过长片段会比较慢。
- 浏览器能不能读取视频取决于本机浏览器的视频解码能力。Mac 上建议使用 MP4/H.264。
- 首次生成会下载 ffmpeg.wasm 运行组件，加载完成后再开始转换。
- 批量转换会按队列逐个处理，视频越多、片段越长，等待时间越长。
